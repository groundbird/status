How to use TOD class (todlib.py)

TOD クラスは時系列データを扱うクラスです。使用する際は、ahiru:/home/hikaru/pythonlibs/todlib/todlib.py にあるものをコピーして使うか、

$ git clone https://github.com/i-hikaru/todlib.git

として todlib.py をダウンロードしてください。folk は歓迎します。

TOD クラスは以下のメソッドを持ちます。

todlib.ts(skip=1)
TOD のタイムスタンプを返すメソッド。 
Parameters:
            skip: int
            指定した整数だけデータを間引きます

Returns: 
            out: numpy.ndarray
todlib.dates(skip=1, tz=pytz.timezone('Asia/Tokyo'))
TOD の時間情報を datetime モジュールの形式で返すメソッド。 
Parameters:
            skip: int
            指定した整数だけデータを間引きます
            tz: pytz.tzfile
            pytz モジュールのタイムゾーンを指定します
            
Returns:
            out: numpy.ndarray
todlib.vals(skip=1)
TOD のすべての値を返すメソッド。 
Parameters:
            skip: int
            指定した整数だけデータを間引きます

Returns:
            out: numpy.ndarray
todlib.get_channel_vals(channel, skip=1)
TOD の各チャンネルの値を返すメソッド。 
Parameters:
            channel: int
            チャンネル番号を指定します
            skip: int
            指定した整数だけデータを間引きます

Returns:
            out: numpy.ndarray
todlib.gentod(skip=1, colslabel=None)
TOD を pandas モジュールの DataFrame 形式で返すメソッド。 
Parameters:
            skip: int
            指定した整数だけデータを間引きます
            colslabel: list
            各チャンネルをラベリングします

Returns:
            out: Pandas.core.frame.DataFrame
todlib.gentod_all(direcroty=None, extension='dat', colslabel=None)
イニシャライズしたファイルの属するディレクトリにあるすべての TOD を pandas モジュールの DataFrame 形式で返すメソッド。 
Parameters:
            direcroty: str
            データファイルが属するディレクトリを指定できます
            extension: str
            データファイルの拡張子を指定できます
            colslabel: list
            各チャンネルをラベリングします

Returns:
            out: Pandas.core.frame.DataFrame

Notes

TOD クラスは使用するデータのある path と使用する列を引数にとってオブジェクトをつくります。ここでは、LakeShore 218 で取得した 8 チャンネルの温度データを例に TOD クラスの使い方を説明します。

Example

LakeShore の温度データ（/home/gb/public_html/gbmonitor/temp/data/lastest）は 10 列の値（時刻, タイムスタンプ, チャンネル 1 の温度, ..., チャンネル 8 の温度）からなります。TOD クラスは 1 列目にタイムスタンプ、2 列目以降をそのときの値（この場合は温度）となるようにデータを指定します。いまの場合は 1 列目以降の列を usecols に指定します。

In [1]: 
%matplotlib inline
import matplotlib.pyplot as plt
import matplotlib.dates as md
import pytz
import pandas as pd
import todlib

tod = todlib.TOD('/home/gb/public_html/gbmonitor/temp/data/lastest', usecols=range(1, 10))

各チャンネルの全期間の温度をプロットするには以下のようにします。

In [2]: 
fig, ax = plt.subplots()
for i in range(8):
    ax.plot(tod.dates(10), tod.vals(10)[:,i], label='ch. '+str(i))
ax.xaxis.set_major_formatter(md.DateFormatter('%m/%d\n%H:%M', tz=pytz.timezone('Asia/Tokyo')))
ax.set_ylabel('Temperature [K]')
leg = ax.legend(loc='best')
leg.get_frame().set_alpha(0.5)
ax.grid()
plt.show()

todlib.gentod() を使うと、同様のプロットがより簡単に描けます。

In [3]: 
tod.gentod(10, ['ch. '+str(i) for i in range(8)]).plot()
plt.show()

todlib.gentod() は pandas の DataFrame を返すので、plot() に限らず、その他色々なメソッドを使うことができます（ref., ）。

ある特定の期間をプロットするには以下のようにします。

In [4]: 
tod.gentod(colslabel=['ch. '+str(i) for i in range(8)])['2014-10-25-0310':'2014-10-25-0320'].plot()
show()

TOD クラスをイニシャライズしたときに引数にとったファイルと同じ階層にある他のファイル（同じデータ構造のファイル）とデータをつなぎあわせて、ひとつの大きい TOD を作りたいときは、todlib.gentod_all() を使います。

In [5]: 
todAll = tod.gentod_all(colslabel=['ch. '+str(i) for i in range(8)])
print todAll
